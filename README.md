
# Notatki

z widokami oraz api.

## Views
```
List    | /notes
Create  | /notes/create
Details | /notes/details/{note}
Edit    | /notes/edit/{note}
```
## Api
```
GET    | api/notes
POST   | api/notes
GET    | api/notes/{note}
PUT    | api/notes/{note}
DELETE | api/notes/{note}
```
